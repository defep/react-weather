## Set-up

Install dependencies and run the app.

```
npm install
npm start
```

Create .env file at the root level with the variable REACT_APP_API_KEY and set your [**Open Weather API Key**](https://openweathermap.org/api).

`REACT_APP_API_KEY=YOUR_API_KEY`
